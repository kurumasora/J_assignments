# cording

## 概要
このリポジトリは、学校の課題を管理するためのモノレポです。

## 4J ディレクトリ概要
`4J` は 4年次（J4）の課題・実験データをまとめたディレクトリです。主に以下の内容を含みます。

- `4J/algorithm`: アルゴリズム関連の課題（C言語の実装、課題資料、レポート下書きなど）
- `4J/arc`: ディジタル回路/アーキテクチャ関連の課題（VHDLソース、テストベンチ、提出資料）
- `4J/experiment`: 実験科目の課題（R解析、アセンブリ、PLC/PWM関連資料・コード・成果物）

必要に応じて `5J` などの新しい年度ディレクトリを追加し、同じ方針で管理します。
